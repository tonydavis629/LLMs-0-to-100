## Self-Supervised Learning

Pretraining means learning from broad raw text **before** specializing the model for any particular task. The trick is where the labels come from.

:::columns cols="2" gap="34px"
**Supervised learning**

Needs humans to label data: each image tagged "cat", each review tagged "positive". Labels are expensive and scarce.
+++
**Self-supervised learning**

The label is already in the text: it is just the **next token**. No human annotation. Every sequence of length $T$ yields $T$ training examples &mdash; each prefix predicts the token that follows it.
:::

This is why pretraining can consume trillions of tokens: the supervision is free, generated by the data itself.

A pretrained model is usually called a **base model**. It has absorbed the statistical structure of text &mdash; grammar, facts, style, code patterns &mdash; but it has **not** been taught to behave like a helpful assistant. That alignment step is Module 6.

## "Just" Predicting the Next Token

Next-token prediction looks almost trivially simple. At scale, it is not: to push the loss lower, the model is **forced** to learn the structure that makes text predictable.

:::columns cols="2" gap="30px"
To reliably continue text, a model must absorb:

- **Grammar and syntax** &mdash; to keep sentences well-formed
- **Facts about the world** &mdash; "the capital of France is ___"
- **Style and register** &mdash; legal text, poetry, code comments
+++
- **Code structure** &mdash; balanced brackets, valid identifiers
- **Task formats** &mdash; question/answer, translation, summaries
- **Implicit reasoning** &mdash; arithmetic, deduction, multi-step chains

:::

The compression pressure of one simple objective, applied to enormous and diverse data, is what produces broad capability.

## A Striking Side Effect: In-Context Learning

A large enough base model can infer a task from **examples in the prompt**, even though it was never trained on that task format:

:::columns cols="2" gap="30px"
```text
sea -> mer
sky -> ciel
cat -> chat
dog ->
```
+++
The model was trained only to continue text. Yet shown a few `english -> french` pairs, it continues the **pattern** and answers `chien`.

This is **few-shot / in-context learning**: an emergent behavior of large-scale pretraining, not an explicitly trained capability.
:::

:::note
We return to in-context learning and prompting in Module 10. For now, note that it appears for free as a consequence of scale &mdash; nobody designed a "few-shot mode" into the training objective.
:::
